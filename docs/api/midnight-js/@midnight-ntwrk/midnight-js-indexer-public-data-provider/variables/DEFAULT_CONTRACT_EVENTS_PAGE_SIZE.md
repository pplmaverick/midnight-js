# Variable: DEFAULT\_CONTRACT\_EVENTS\_PAGE\_SIZE

> `const` **DEFAULT\_CONTRACT\_EVENTS\_PAGE\_SIZE**: `100` = `100`

Default page size for PublicDataProvider.queryContractEvents when the
caller omits `page.limit`. Applied explicitly by the provider so the page
size is documented and stable, never an undocumented server default.
